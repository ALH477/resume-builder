# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-01-30

### Added
- Initial release of Resume Builder
- GTK3-based graphical interface
- Support for multiple resume sections:
  - Personal header (name, subtitle)
  - Contact information (website, email, phone)
  - Work experience with bullet points
  - Projects with descriptions
  - Education and certifications
  - Skills categorization
- Save/Load functionality using JSON format
- HTML export with professional styling
- Print-ready output optimized for letter-size paper
- Preview functionality
- Nix flake for easy installation and development
- Apache 2.0 license
- Comprehensive documentation

### Features
- Clean, modern timeline-based resume design
- Responsive layout with sidebar
- Professional typography using Open Sans
- Decorative timeline elements
- Easy-to-use dialog boxes for data entry
- Remove buttons for editing entries
- Form validation and data persistence
